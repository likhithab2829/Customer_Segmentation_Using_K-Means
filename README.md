# Customer Segmentation System

## Project Overview
This project groups customers into different segments based on purchasing behavior using K-Means Clustering.

## Objective
To help businesses identify different categories of customers for targeted marketing.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm
- K-Means Clustering

## Features
- Automatic customer grouping
- Data visualization
- Cluster analysis
- Marketing insights

## Input Parameters
- Customer Income
- Spending Score

## Output
- Customer Clusters

## Project Workflow
1. Collect customer data
2. Apply K-Means algorithm
3. Create clusters
4. Visualize customer groups

## Future Improvements
- Dynamic cluster selection
- Larger datasets
- Interactive dashboard

## Conclusion
Customer segmentation helps businesses improve marketing strategies and customer engagement.
